#Backup

1) Open Mysql Workbench
2) Click on Local instance MySQL80
3) click on Data Export on left panel
4) In Tables to Export section, select the database you want to export
5) In Export Options section, select Export to Dump Project Folder. Dont change the path
6) Click Start Export
7) Go bact to Object selection tab, select the path and paste it in MY PC to check the restored file.

#Restore

1) Click on Data Import/Restore
2) In Import Option section, Paste the path of the restored files used in backup and Load Folder Contents
3) In Select Database Objects to Import section, select the data base you want to restore. Click on the database and and select the tables
4) Click Start Import
